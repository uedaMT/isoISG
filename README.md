# isoISG

How alternative splicing events, especially dynamic changes over time, affect the function of ISGs remains poorly understood. We generated a novel isoform annotation for IFN-Î±-stimulated B-cells, called isoISG (isoforms of Interferon-Stimulated Genes), using two long-read sequencing platforms (PacBio IsoSeq and Nanopore).


A UCSC browser is available
https://genome.ucsc.edu/s/UEDA/isoISG


The annotation files generated in LCL
